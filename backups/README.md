# Backups Folder Layout

This folder keeps backup-related files organized while keeping the repo root clean.

## Tracked paths

- `legacy/`
  - One-off legacy files intentionally kept in git for reference.
  - Current item: `legacy/config.conf.jpfont.backup`

## Local-only paths (gitignored)

- `snapshots/`
  - Generated by `scripts/snapshot-create.sh`
  - Contains timestamped snapshot folders, `latest` symlink, and `LATEST_PATH.txt`

## Notes

- Snapshot archives are local artifacts and are not meant to be committed.
- Keep only long-lived reference backups in `legacy/`.
